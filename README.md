# MBProgressHUD-LJ

MBProgressHUD 的一个分类，调用更加方便。

支持横竖屏，以及 iPad 的适配。

## 用法

    //将 MBProgressHUD+LJ 文件夹拉到项目中。
    
    #import "MBProgressHUD+LJ.h"
    
    // 显示一个带遮罩的 HUD
    [MBProgressHUD showMessage:@"请骚等一会儿"];
    
    // 显示一个 success 的 HUD
    [MBProgressHUD showSuccess:@"登录成功"];
    
    // 显示一个 error 的 HUD
    [MBProgressHUD showError:@"服务器被外星人搬走了o(╯□╰)o"];
    
    // 隐藏所有 HUD
    [MBProgressHUD hideHUD];
